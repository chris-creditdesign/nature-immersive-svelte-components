# Nature Immersive Svelte Components

Welcome to the `nature-immersive-svelte-components` storybook.

This is a collection of [svelte](https://svelte.dev/) components forms the design system used to build [@nature](https://twitter.com/nature) immersive articles, such as [_Nature_â€™s 10](https://www.nature.com/immersive/d41586-020-03435-6/index.html).

## Installation

To install the components from npm:

```bash
npm install --save-dev nature-immersive-svelte-components
```

## Useage

Import each component into a svelte file with:

```js
import { Card, CardBox, etc... } from "nature-immersive-svelte-components";
```

To save from starting a project from scratch, you can use the [`nature-immersive-article-template`](https://github.com/chris-creditdesign/nature-immersive-article-template). Either download the files from this repo or use [`degit`](https://github.com/Rich-Harris/degit):

```bash
mkdir new-article
cd new-article
npx degit chris-creditdesign/nature-immersive-article-template
```

## Load CSS

The static CSS files can be concatenated to form a base style sheet:

```bash
cat node_modules/nature-immersive-svelte-components/static/css/fonts.css \
	node_modules/nature-immersive-svelte-components/static/css/custom-props.css \
	node_modules/nature-immersive-svelte-components/static/css/light-theme.css \
	node_modules/nature-immersive-svelte-components/static/css/index.css \
	> public/css/index.css
```

You can choose to include either `light-theme.css` or `dark-theme.css`.

From then on style can be customised by passing props to existing svelte components or as style blocks within your own components.
